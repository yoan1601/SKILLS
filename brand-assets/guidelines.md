# Visual Identity System

<aside>
🎨

**Purpose:** Visual consistency that signals executive-level clarity: minimal, structured, system-oriented.

</aside>

### Visual foundations (premium + minimal)

**Color palette (guideline)**

- Primary: Deep Navy (#0B1F3B)
- Primary: Slate (#1F2937)
- Neutral: Off-White (#F8FAFC)
- Neutral: Light Gray (#E5E7EB)
- Accent: Signal Green (#10B981) *used sparingly for “go / shipped / verified”*

**Typography (guideline)**

- Headline: Inter / SF Pro Display (Semibold)
- Body: Inter / SF Pro Text (Regular)
- Rule: one typeface family, two weights, strict hierarchy

**Image direction**

- Real environments, natural light, minimal background.
- Prefer diagrams, systems, and operating visuals over stock “AI” imagery.

**Post style rules**

- High whitespace
- 1 idea per visual
- Strong typographic hierarchy
- Never more than 2 accent elements per slide

### LinkedIn visual standards

**Banner concepts**

- “Turning AI ambition into operational reality.” + minimal system diagram (Data → Decision → Workflow → Feedback)

**Headshot style**

- Neutral background, direct posture, minimal editing; executive presence.

**Carousel grid rules**

- 1080×1350 (portrait)
- 64px margins (safe area)
- 8–10 slides maximum
- Slide 1: claim; Slide 2–8: framework; last slide: operating rule + one question

### Presentation standards

**Slide style**

- Title + 3 bullets maximum per slide
- Use divider lines; avoid dense paragraphs

**Diagram conventions**

- Boxes = components; arrows = flows; labels = verbs (e.g., “Validate”, “Route to review”)
- Always include: failure mode + control (at least one)
